Login

Wenn Sie Ihre Zugangsdaten und die Internetadresse für das Sherlock-Holmes-Systems bekommen haben, können Sie sich einloggen.
Rufen Sie die URL des Sherlock-Holmes-Systems (z.B. https://...) in Ihrem Browser auf und geben Sie Benutzernamen und Kennwort in das Dialogfeld „Login“ ein.
Bestätigen Sie anschließend mit „Login“.


Noch nicht registriert?

Wenn Sie noch nicht im System registriert sind, klicken Sie auf „Registrieren-Button“, der Se zum Registrierungsformular führt.

Das Registrierungsformular ist folgendermaßen aufgebaut:

  1. Benutzername festlegen
  2. E-Mail
  3. Neues Passwort
  4. Passwort wiederholen

Bestätigen Sie zum Abschluss die Schaltfläche „Jetzt registrieren“.

Nachdem Sie registriert sind, erhalten Sie eine Begrüßungsnachricht. Wenn Sie dort die Schaltfläche „Anmelden“ bestätigen, werden Sie automatisch angemeldet.


Passwort vergessen?

Wenn Sie Ihr Passwort vergessen haben, können Sie sich eine neue Aktivierungs-E-Mail zusenden lassen. Folgen Sie dem Verweis „Passwort vergessen?“ im Anmeldeportlet.
Dieser Verweis führt Sie zu einem Formular, auf dem Sie Ihren Benutzernamen oder E-Mail-Adresse eingeben und anschließend die Schaltfläche E-Mail anfordern bestätigen müssen.
Sie erhalten daraufhin eine E-Mail, die einen Verweis zu einer Seite, auf der Sie ein neues Passwort eingeben können, enthält.
